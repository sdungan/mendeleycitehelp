User Guide

Online Help for Mendeley Cite

Welcome to the Mendeley Cite online user guide.

This plugin lets you insert citations and bibliographies from your Mendeley library into Microsoft® Word documents. Mendeley Cite works with Microsoft Word 2016 and later, including Microsoft Office 365.

If you're new to Mendeley, click Getting Started for installation instructions and an overview of the plugin.
