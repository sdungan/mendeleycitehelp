Frequently Asked Questions

What's the difference between a reference and a citation?

A "citation" is an in-text element that indicates the preceding text is a quote or paraphrase from an outside source. A "reference" is a full source notation that appears in a list (i.e. a bibliography) at the end of a document. Each in-text citation should refer to a reference in the bibliography.

What's the difference between a footnote and an endnote?

A "footnote" is a note or citation at the bottom of a page that's referred to somewhere in the text on the same page. An "endnote" serves the same function as a footnote except they are gathered in a section at the end of the paper instead of at the bottom of pages. Each citation in a footnote or endnote should also refer to a reference in the bibliography.

How do I make citations and see articles in my library at the same time?

There are two ways to keep your reference library easily accessible while working in Word:

  • Option 1—With a larger screen you can narrow each application window to occupy one half of the screen.
  • Option 2—Use the Alt+Tab keyboard shortcut to quickly switch between each application.

Tip: Access your Mendeley library online with a mobile device to quickly refer to it while writing on a desktop computer.

Can I change an in-text citation so that it's just the year without the author?

Yes! You can suppress the author name of an in-text citation by editing the citation in Mendeley Cite.

  1. With Mendeley Cite open, click the citation with the author name you want to suppress. The citation details display in the Mendeley Cite panel.
  2. Click the reference pill-box. The Edit Reference window displays.
  3. Click the Suppress author checkbox and Save changes.

The author name is removed from the citation. Go to the Citations page for full instructions.

How do I add page numbers to an in-text citation?

You can add page numbers (or other items including chapters, volumes, and issues) to an in-text citation by editing the citation in Mendeley Cite.

  1. With Mendeley Cite open, click the citation with the author name you want to suppress. The citation details display in the Mendeley Cite panel.
  2. Click the reference pill-box. The Edit Reference window displays.
  3. Enter page numbers in the Value field next to the Page label and Save changes.

The page numbers are added to the citation. Go to the Citations page for full instructions.

How do I uninstall Mendeley Cite?

Uninstall Mendeley Cite from within Microsoft Word.

  1. Go to Word's References menu.
  2. Right-click the Mendeley Cite button and select Remove Add-in. A confirmation window displays.
  3. Click Yes to delete Mdneley Cite from Word.
